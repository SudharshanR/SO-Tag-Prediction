1. Combine the files in 'Original_files/' using 'python_code/preprocess.py' to get inter-Output.csv

2. Generate 'inter_test.csv' and 'inter_train.csv' using 'python_code/random_sampling.py'

3. Generate 'inter_final_test.csv' and 'inter_final_train.csv' by joining tags using 'preprocess.py'

4. Run 'python_code/bashscript' to generate 'directory.csv' and 'directirytest.csv', which contains the distinct tag-counts for 'inter_final_test.csv' and 'inter_final_train.csv' respectively

5. Use 'python_code/preprocess.py' to generate 'final_test.csv' and 'final_train.csv'. These files are generated by removing rows from  'inter_final_test.csv' and 'inter_final_train.csv' whose tag is not in the top 20.

6. Run 'python_code/baseline.py' to form a dictionary of postID and tag for the final_train.csv file, read top 5 tags from trainingset.csv file and find accuracy of random prediction and topTag prediction.

7. Run 'python_code/createmap.py' to generate 'dict.csv', which contains count of all distinct words.

8. Run 'python_code/GenerateClassFiles.py' to generate separate files for top 20 tags.

9. Run 'python_code/createmap.py' to generate the classifier for each of the top 20 tags and store the classifier as a pickle file.

10. Run 'python_code/testSetEval.py' to find prediction of test set.
